## 版本说明

| 版本 | 日期 | 说明 |
| :--- | :--- | :--- |
| v1.0 | 2026-03 | 初始需求规划（理想功能全集） |
| v2.0 | 2026-03 | 结合 v1.0 桌面客户端开发成果更新，区分已实现 / 待规划功能 |

## 产品概述

- **产品名称**：云笔记
- **定位**：面向日常办公的智能笔记助手，参考有道云笔记、印象笔记
- **技术栈**：Electron v33 + 原生 HTML/CSS/JS（无框架）
- **运行平台**：Windows 桌面客户端（exe），同时支持浏览器直接打开
- **数据存储**：localStorage 本地持久化
- **打包方式**：electron-packager，产出约 269MB

## 十、已知问题与优化方向

| 类别 | 描述 | 建议 |
| :--- | :--- | :--- |
| 数据安全 | localStorage 数据量有限（约 5-10MB），无法存储大量笔记 | 迁移至 IndexedDB 或 SQLite |
| 图片存储 | 本地图片以 base64 嵌入，笔记体积膨胀快 | 改为文件系统存储 + 引用路径 |
| 搜索性能 | 全文搜索为内存遍历，笔记多时性能下降 | 引入全文索引（如 FlexSearch） |
| 安装包体积 | 269MB（Electron 运行时占大头） | 考虑增量更新或 electron-builder nsis |
| 打印适配 | PDF 导出对复杂表格和代码块的分页可优化 | 引入分页控制 CSS |
